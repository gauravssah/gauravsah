# Image folder

Put the site images in this folder using these names:

- profile-portrait.jpg - Main profile image used across navbar and about section
- hero-particle.png - Optional decorative hero asset if you want to replace the built-in 3D-only look
- project-animalslover.png - Optional screenshot for the main project card
- og-cover.png - Optional social sharing image
- favicon.png - Optional browser icon

For certifications, place images in `public/images/certificates` with these names:

- certificate-1.jpg
- certificate-2.jpg
- certificate-3.jpg
- certificate-4.jpg

The app currently uses `profile-portrait.jpg` and automatically maps certificate cards to `certificate-1.jpg`, `certificate-2.jpg`, and so on.
